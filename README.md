# ADOMC - HEREUSTIQUE MIN-CONFLICT


Dans cette application, chaque point d’accès Wi-Fi représente une variable du problème. Les canaux Wi-Fi disponibles représentent les valeurs possibles. Les liaisons entre les points d’accès représentent les zones d’interférence. Lorsqu’un point d’accès utilise le même canal qu’un autre point d’accès voisin, un conflit est détecté. L’heuristique Min-Conflicts sélectionne alors un point d’accès en conflit et lui attribue le canal qui minimise le nombre total de conflits. Le processus est répété jusqu’à obtenir une configuration optimale ou proche de l’optimal.
